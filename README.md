CarWash - CRUD-приложение для управления автомойкой 
Полноценное веб-API для автомотизации работы автомойки 

Чтобы использовать нужно .NET 8 SDK и PostgreSQL

После запуска проекта откройте в браузере адрес http://localhost:5226/swagger — перед вами откроется интерфейс Swagger. Через него вы сможете управлять всеми данными.

Как добавить клиента
1)В разделе Clients найдите строку POST /api/clients.
2)Нажмите кнопку «Try it out» справа.
3)В поле ниже появится пример JSON. Измените значения:
    "fullName" — напишите ФИО клиента, например "Иванов Иван Иванович"
    "phoneNumber" — номер телефона, например "+79991234567"
4)Нажмите «Execute».
5)В ответе вы увидите статус 201 Created и данные нового клиента — обязательно запомните его Id (он понадобится дальше).
Как посмотреть всех клиентов
1.В том же разделе Clients найдите GET /api/clients.
2.Нажмите «Try it out», затем «Execute».
3.В ответе вы получите список всех клиентов с их Id, ФИО и телефонами.
Как добавить автомобиль клиенту
1.Перейдите в раздел Cars.
2.Найдите POST /api/cars и нажмите «Try it out».
3.В JSON укажите:
    "brand" — марка машины, например "Toyota"
    "model" — модель, например "Camry"
    "licensePlate" — госномер, например "А123БВ777"
    "clientId" — Id клиента, который вы получили при создании (в формате "a1b2c3d4-...")
Нажмите «Execute» — машина будет привязана к клиенту.
Как посмотреть, какие машины у клиента
1.В разделе Cars найдите GET /api/cars/client/{clientId}.
2.Нажмите «Try it out».
3.В поле clientId введите Id нужного клиента.
4.Нажмите «Execute» — вы увидите список всех его автомобилей.
Как добавить услугу (например, "Мойка кузова")
1.Перейдите в раздел Services.
2.Найдите POST /api/services → «Try it out».
3.Укажите:
    "name": "Мойка кузова"
    "description": "Полная мойка снаружи"
    "price": 500.00
4.Нажмите «Execute» — запомните Id услуги.
Как создать заказ
1.Перейдите в раздел Orders.
2.Найдите POST /api/orders → «Try it out».
3.В JSON укажите:
    "clientId" — Id клиента
    "carId" — Id автомобиля (из ответа при создании машины)
    "serviceIds" — массив с Id услуг, например: ["s1...", "s2..."]
4.Нажмите «Execute».
5.В ответе вы увидите заказ с полем "totalAmount" — сумма рассчитана автоматически!
Как посмотреть заказы
    Чтобы увидеть все заказы: GET /api/orders
    Чтобы увидеть только завершённые: GET /api/orders/status/Completed
    Чтобы увидеть заказы за период: GET /api/orders/date-range → укажите startDate и endDate в формате 2025-11-14
Как удалить клиента
1.В разделе Clients найдите DELETE /api/clients/{id}.
2.Нажмите «Try it out».
3.В поле id введите Id клиента.
4.Нажмите «Execute».
5.Клиент и все его автомобили будут удалены автоматически (каскадное удаление).
